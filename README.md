AI Resume Builder is a web app that helps users create, edit, and improve resumes using AI. Upload an existing resume or build one from scratch with smart suggestions and clean formatting.

Features

Create resumes from scratch
AI-powered content generation & editing
Upload and improve existing resumes
ATS-friendly formatting
Real-time preview while editing
Tech Stack

Frontend: React.js, Tailwind CSS
Backend: Node.js,Express,API
Use Cases

Students building their first resume
Job seekers improving existing resumes
Professionals updating resumes quickly
